# Project

## What This Is
An industrial-grade Multi-Agent Orchestrator using the Model Context Protocol (MCP) as its universal connection layer. Establishes a standardized "system nervous" that allows autonomous agents to perceive, reason, and act within a secure enterprise environment, integrating specific course materials as the foundational "Theory-to-Action" bridge.

## Core Value
To create a universal connection layer for autonomous agents, enabling them to perceive, reason, and act within a secure enterprise environment via the MCP, directly applying Fargin Curriculum theory.

## Current State
Initial setup for Milestone M001: Core Orchestrator and MCP Foundation. The project structure and planning artifacts for M001 are being initialized.

## Architecture / Key Patterns
Decoupled Client-to-Server architecture with an Orchestrator (Host), Connector (Client), and Specialist Servers (MCP Servers). Implements ReAct pattern (Perception → Reasoning → Action) using Chain-of-Thought (CoT). Emphasizes deterministic workflows, privacy-first execution (on-device inference), and secure validation (JSON Schema/Pydantic).
